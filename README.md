# Pokemon Info Fetcher

A Python program that connects to the PokeAPI and retrieves real-time information about any Pokemon.

## What it does:
- Takes a Pokemon name as input
- Connects to the PokeAPI
- Fetches real data about that Pokemon
- Displays name, ID, height, and weight

## How to run:
1. Install requests: `pip install requests`
2. Run `python pokemon-api.py`
3. Enter any Pokemon name (try "pikachu", "charizard", "mewtwo")

## What I learned:
- Making API requests with `requests` library
- Parsing JSON data
- Error handling for failed requests
- Working with real-time internet data

## API Used:
[PokeAPI](https://pokeapi.co/) - Free Pokemon data API
